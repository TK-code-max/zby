TVBOX直播源
来自potplayer直播源m3u文件
在这里可以直接进行直播地址引用

[各版本tvbox在此下载](https://pan.quark.cn/s/fe07c0af72de#/list/share)

takagen99 [维护封装的版本在此下载](https://github.com/j4Uq/TVBoxOSC/releases)

关于各种源维护的比较全的是ngo5 狸花 [链接在此](https://github.com/ngo5/IPTV)

如果实在苦于各种源的切换，可以考虑基于web浏览器壳子的U桃，[链接](https://www.utao.tv/)

智能电视优先使用arm64embi版本
各种盒子优先使用arm64 java版本

如果电视安装java版本提示无存储权限，可改安装python版

如果运行提示jar文件无法存储或为空，可设置安全DNS，可以挨个尝试，然后彻底推出盒子，重新进入，直到找到最快的安全DNS。安全DNS有时会失效，则需要尝试更换其它的。
